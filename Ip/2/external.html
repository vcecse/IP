STYLE SHEET

  • INTERNAL STYLE SHEET

    • Internal style sheet may have higher precedence than external style sheets.this is determind bythe order that the external style sheets are loaded.

  • EXTERNAL STYLE SHEET

    • An external style sheet is a separate file where you can declare all the styles that you want to use on your website.Then link the external style sheet from all your HTML pages,means you only need to set the styles for each element once.If you want to update the style of your website,you only need to do it in one place.


  • EMBEDDED STYLE SHEET

    • Embedded style sheet refers,you embed style sheet information in to an HTML document using the tags in the head of your document.
